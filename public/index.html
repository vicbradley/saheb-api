Saheb API

REST API Website Saheb

Visit Saheb

Saheb API adalah sebuah REST API yang dapat Anda gunakan untuk mengumpulkan data produk dan toko dari website Saheb

Dokumentasi

Data Produk

Get: Semua Produk

fetch('https://saheb-api.vercel.app/products')

./then(res=>res.json())

.then(json=>console.log(json))

Get: Produk Berdasarkan Id

fetch('https://saheb-api.vercel.app/products/4eVkzBbSMRoK36piFYa7')

./then(res=>res.json())

.then(json=>console.log(json))

Get: Produk Berdasarkan Keyword Search

fetch('https://saheb-api.vercel.app/products/search/harness')

./then(res=>res.json())

.then(json=>console.log(json))

Data Toko

Get: Toko Berdasarkan Id

fetch('https://saheb-api.vercel.app/stores/saheb')

./then(res=>res.json())

.then(json=>console.log(json))

Get: Produk Berdasarkan Toko

fetch('https://saheb-api.vercel.app/stores/saheb/products')

./then(res=>res.json())

.then(json=>console.log(json))

Get: Produk Toko Berdasarkan Keyword Search

fetch('https://saheb-api.vercel.app/stores/saheb/products/search/pedigree')

./then(res=>res.json())

.then(json=>console.log(json))
